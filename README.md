# Umbra Sapiens

**Version:** 0.17d Istanbul (Not Constantinople)

A Crusader Kings II mod that extends the Umbra Spherae-Reborn experience with additional content and features.

## Description

Umbra Sapiens is a comprehensive modification for Crusader Kings II that builds upon the Umbra Spherae-Reborn framework. This mod adds extensive historical and alternate history content spanning the globe.

## Requirements

- **Crusader Kings II** (base game)
- **Umbra Spherae-Reborn** (dependency mod)

## Installation

1. Ensure Crusader Kings II is installed
2. Install the Umbra Spherae-Reborn mod (required dependency)
3. Clone or download this repository to your CK2 mod folder:
   - Windows: `Documents/Paradox Interactive/Crusader Kings II/mod/`
   - Linux: `~/.paradoxinteractive/Crusader Kings II/mod/`
   - Mac: `~/Documents/Paradox Interactive/Crusader Kings II/mod/`
4. Place the `umbra_sapiens.mod` file in the mod directory
5. Place the `umbra_sapiens` folder in the mod directory
6. Launch Crusader Kings II and enable "Umbra Sapiens" in the launcher

## Features

- **Global Province Coverage**: Extensive provinces across all continents
- **Custom Graphics**: Unique coats of arms and flags for various cultures including:
  - Japanese dynasties and patterns
  - Mesoamerican (Aztec, Mayan, Inca)
  - Uralic cultures
  - Native American nations
  - Pacific and Australian indigenous groups
- **Historical Events**: Custom events and decisions
- **Cultural Diversity**: Support for numerous culture groups and religions
- **Custom Localisation**: Localized text for enhanced immersion

## Mod Structure

```
umbra_sapiens/
├── common/         # Game rules, cultures, religions, buildings, etc.
├── decisions/      # Custom decision events
├── events/         # Historical and scripted events
├── gfx/           # Graphics assets (flags, coats of arms, interface)
├── history/        # Historical province and character data
├── interface/      # UI modifications
├── localisation/   # Translations and text
└── map/           # Map data and definitions
```
